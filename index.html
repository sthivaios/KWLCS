KWLCS is discontinued

KWLCS was an experiment to demonstrate how devices can be controlled using a simple website. KWLCS has been discontinued as of June 15, 2024. We are working on a similar new experiment, AWLCS.

About AWLCS

AWLCS stands for Advanced Web LED Control Service.

AWLCS implements real-time indicators that show the state of each LED, as well as API rate limiting, rate limiting on the web interface, and authentication, including MFA.

ALWCS is expected to be publicly available on July 30, 2024.

An early access program may be available to some users after July 10.

Learn how to register for AWLCS at awlcs.kioydiolabs.org
